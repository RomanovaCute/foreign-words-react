# Vocaby - приложение для изучения английских слов
### Demo: http://romanovacute.github.io/foreign-words-react/
### Getting started
1. ``git clone https://github.com/RomanovaCute/foreign-words-react.git``
2. ``npm install``
3. ``npm start``

### Technologies:
+ React
+ MobX
+ react-router
+ styled-components
